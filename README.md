# Python Basics: Assignment 1 

## 📝 Overview
This repository contains a Jupyter Notebook (`Assignment(1).ipynb`) featuring a series of basic Python programming exercises. It is designed to demonstrate fundamental Python concepts such as loops, conditional statements (`if-else`), mathematical operations, and the use of imported modules.

##  Features & Exercises

The notebook includes the following four Python programs:

1. **Prime Number Checker**
   * A script that takes a user's input and calculates whether the given number is a prime number by checking its positive divisors.
2. **Multiplication Math Game**
   * A short interactive game that uses the `random` module to generate two numbers between 1 and 10. It prompts the user to calculate the product and checks if their answer is correct.
3. **Number Squares (Even Numbers)**
   * A script utilizing a `for` loop to iterate through a specific range (100 to 200). It filters for even numbers and prints out the calculated square of each.
4. **Hospital Bill Calculator**
   * A practical, real-world scenario script that calculates a final hospital bill based on:
     * Room charges (₹2,000/day)
     * Doctor consultation fee (₹1,500 flat)
     * Lab test charges (₹300/test)
     * Medicine charges
   * It also includes logic to apply a 10% discount if the total bill exceeds ₹10,000. *(Note: This section includes the foundational logic for the calculation based on a specific patient scenario).*
